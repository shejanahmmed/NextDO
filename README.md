# NextDO

NextDO is a minimal to do app that helps you stay organized and on time. You can create tasks set reminders and get notified exactly when you need to take action. The app is built with Java and designed to be smooth fast and clean.

## Features
- Add and manage tasks easily  
- Set reminders for specific times  
- Get timely notifications  
- Simple and minimal UI  
- Smooth performance  
- Built with Java for Android  
- Highly customizable  
- Gesture support
